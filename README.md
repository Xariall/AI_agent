# AI Agent Test Task

## Описание проекта

Это учебный проект, который демонстрирует связку:
- **FastAPI** как HTTP‑сервис.
- **LangGraph** как агентный оркестратор.
- **FastMCP** как MCP‑сервер с инструментами работы с товарами.

Проект нужен, чтобы показать полный путь запроса:
1. Пользователь отправляет текстовый запрос в API.
2. Агент (Mock LLM) определяет намерение и вызывает нужный инструмент.
3. Инструменты обращаются к MCP‑серверу (через stdio‑подпроцесс) и возвращают данные.
4. API отдает финальный ответ пользователю.

Таким образом демонстрируется интеграция агентного пайплайна с MCP‑инструментами и обычным веб‑API.

## Архитектура

- `mcp_server/` — MCP‑сервер с инструментами:
  - список товаров
  - получение товара по id
  - добавление товара
  - статистика по ценам
- `agent/` — LangGraph агент с mock‑маршрутизацией
- `app/` — FastAPI сервис
- `data/products.json` — локальное хранилище товаров

## Запуск через Docker Compose

```bash
docker compose up --build
```

Сервис будет доступен на `http://localhost:8000`.

## API

**Метод:** `POST`  
**Путь:** `/api/v1/agent/query`  
**Тело запроса:** `{"query": "строка запроса"}`

### Примеры CURL

**Показать все товары в категории Электроника**
```bash
curl -X POST http://localhost:8000/api/v1/agent/query \
  -H "Content-Type: application/json" \
  -d '{"query":"Покажи все продукты в категории Электроника"}'
```

**Средняя цена по всем товарам**
```bash
curl -X POST http://localhost:8000/api/v1/agent/query \
  -H "Content-Type: application/json" \
  -d '{"query":"Какая средняя цена продуктов?"}'
```

**Добавить новый товар**
```bash
curl -X POST http://localhost:8000/api/v1/agent/query \
  -H "Content-Type: application/json" \
  -d '{"query":"Добавь новый продукт: Мышка, цена 1500, категория Электроника"}'
```

**Посчитать скидку на конкретный товар**
```bash
curl -X POST http://localhost:8000/api/v1/agent/query \
  -H "Content-Type: application/json" \
  -d '{"query":"Посчитай скидку 15% на товар с ID 1"}'
```
